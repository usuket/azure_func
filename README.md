##Dependencies
- nodejs 8.11.XX or 10.14.XX
- npm install -g azure-functions-core-tools
- https://docs.microsoft.com/ja-jp/cli/azure/install-azure-cli?view=azure-cli-latest

##Init a function
func init

##Create a function
func new

##Run locally 
func start

##Deploy
```

az login
func azure functionapp publish function-app-01
```
